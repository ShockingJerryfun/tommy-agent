# DREAMS

Tommy Agent 的长期方向是成为一个可本地运行、可审计、可扩展的 Agent 工作台。

## 项目目标

- 保持本地优先：会话、检查点、记忆和技能提案默认保存在用户自己的机器上。
- 保持安全边界清晰：高风险工具调用需要审批，公开提交前需要脱敏。
- 保持体验透明：前端应展示运行过程、工具调用、审批状态和可恢复上下文。
- 保持可扩展：后端工具、Skill、记忆和子 Agent 能以清晰接口逐步增强。

## 后续改进方向

- 增强运行时数据的导入、导出和备份能力。
- 为公开模板和私有运行数据提供更明确的迁移流程。
- 增加端到端测试，覆盖会话创建、流式输出、审批和记忆提案。
